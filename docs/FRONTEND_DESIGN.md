# KM 文件管理系統 - 前端設計規劃

## 設計風格

### 色彩主題
- **簡潔黑白對比**
- 背景：純白 `#ffffff`
- 主要文字：黑色 `#1a1a1a`
- 邊框/分隔線：淺灰 `#e5e5e5`
- 強調/選中：黑色 `#000000`
- 無漸層、無毛玻璃效果

### 設計原則
- ✨ 簡單乾淨
- 📐 大量留白
- 📊 Excel 風格表格
- ⬛ 黑白高對比

---

## 主要頁面：文件清單

**類似 Excel 的可編輯表格**

```
┌────────────────────────────────────────────────────────────────┐
│ 📚 文件管理系統                     [+ 上傳文件]  [匯出 Excel] │
├────────────────────────────────────────────────────────────────┤
│ 部門篩選: [全部 ▼]   狀態: [全部 ▼]   🔍 搜尋...              │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│ ┌──────────────────────────────────────────────────────────┐  │
│ │ 文件編號 │ 文件名稱    │ 部門   │ 分類 │ 版本│ 狀態│ 操作 │
│ ├──────────────────────────────────────────────────────────┤  │
│ │ LAW-001  │ 勞動基準法  │ 法遵部 │ 法規 │ v1  │ ✅  │ ⋮  │  │
│ │ HR-001   │ 員工請假辦法│ 人資部 │ 辦法 │ v2  │ ✅  │ ⋮  │  │
│ │ HR-002   │ 考勤管理辦法│ 人資部 │ 辦法 │ v1  │ 📝  │ ⋮  │  │
│ │ SOP-001  │ 新人報到流程│ 人資部 │ SOP  │ v3  │ ✅  │ ⋮  │  │
│ └──────────────────────────────────────────────────────────┘  │
│                                                                │
│ 顯示 1-10 / 共 25 筆                        [< 1 2 3 >]       │
└────────────────────────────────────────────────────────────────┘
```

### 表格功能
- ✏️ **雙擊編輯**：直接在表格內編輯
- 🔀 **欄位排序**：點擊標題排序
- 🔍 **篩選過濾**：部門、狀態、關鍵字
- ⬇️ **匯出 Excel**：下載為 .xlsx

## 技術架構

| 項目 | 選擇 |
|------|------|
| 框架 | React + Vite |
| 樣式 | Vanilla CSS（黑白簡潔） |
| 表格 | 自建可編輯表格元件 |
| 資料 | 靜態 JSON（或 localStorage）|
| 部署 | GitHub Pages |

## CSS 設計變數

```css
:root {
  /* 黑白對比 */
  --color-bg: #ffffff;
  --color-text: #1a1a1a;
  --color-border: #e5e5e5;
  --color-hover: #f5f5f5;
  --color-active: #000000;
  --color-active-text: #ffffff;
  
  /* 狀態色 */
  --color-success: #22c55e;
  --color-warning: #f59e0b;
  --color-draft: #6b7280;
  
  /* 圓角 */
  --radius: 4px;
  
  /* 間距 */
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
}
```
